Design a complete **web and mobile responsive UI system** for a **Disaster SOS & Rescue Coordination Platform** used during natural disasters.
The system connects **citizens, rescue coordinators, and rescue teams** to report incidents, verify emergencies, assign rescue missions, and coordinate rescue operations.

The interface must be **clean, professional, and emergency-focused**, similar to government emergency systems.

Design style:

* Modern, minimal, professional
* Color palette:

  * Primary: Red (#E53935) for emergency actions
  * Secondary: Blue (#1976D2) for system actions
  * Neutral: white / light grey backgrounds
* Clear typography
* High accessibility
* Mobile-first responsive design
* Use card layouts and map integration
* Include icons for emergency, location, rescue, warning

Create the following **complete screens and flows**.

---

1. USER AUTHENTICATION

Create screens for citizen account registration and login.

Screens:
• Login page
• Register account page
• OTP verification screen
• Forgot password page

UI components:
• Email / phone input
• Password field
• OTP input (6-digit boxes)
• Verify button
• Error messages
• Success confirmation

Flow:
Citizen enters registration info → system checks existing account → send OTP → user enters OTP → account created.
