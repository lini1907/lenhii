# 🌊 Disaster Response Coordination System

A web-based platform that supports disaster monitoring, emergency reporting, and rescue coordination.
The system connects citizens, rescue teams, and government coordinators to respond faster and more efficiently during natural disasters.

---

## 🚨 Project Overview

Natural disasters such as floods, storms, and landslides frequently affect many regions in Vietnam.
This project provides a digital platform that enables real-time communication between citizens and emergency authorities.

The system allows citizens to receive alerts and report emergencies, while coordinators and rescue teams can monitor situations and manage rescue missions.

---

## 👥 User Roles

### 🧑‍🤝‍🧑 Citizen

* View disaster alerts without logging in
* Send SOS emergency reports
* Track rescue progress and updates

### 🧑‍💼 Coordinator

* Monitor alerts and SOS reports
* Create disaster alerts
* Assign rescue missions
* Manage rescue operations

### 🚑 Rescue Team

* Receive assigned missions
* Update rescue status
* Report mission results
